# python3-mecab-neologd-dockerfile

Dockerfile for Python3.6, Mecab, Neologd.

## Getting Started

1. build image  
`$ docker build -t container:1.0 .`
2. run container  
`$ docker run -it -p 8888:8888 -v ~/Desktop/work:/home container:1.0`

You can use `$python3` .
